Transcendence Mods

Transcendence by George Moromisato is the premier space opera of the 2000s. I made some mods for it back in 2016.


Transgenesis II

Transgenesis II is a smart XML editor inspired by command line programs. Use commands to load files, navigate structures, and manage elements and attributes. The program ensures that you create valid XML for the specific domain by only allowing you to create elements and attributes specified by a schema file. Originally developed for the purpose of creating Transcendence mods. Written in C# using MonoGame and SadConsole.


[WIP] ASECII

ASCII Sprite Editing Console II (ASECII) is an ASCII art painter inspired by Aseprite and REXPaint. Written in C# using MonoGame and SadConsole.

CaveX

ASCII-style platformer game for Citrus Hack 2019. Control the mecha X as it descends a cave full of grenadiers. You have no attack, but you can push things around by moving!


ROSE || HACK

ASCII-style platformer game for Rose Hack 2021. Ascend the Great Rose! Plant a ROSE to climb up and HACK the enemy bugs flying around!
