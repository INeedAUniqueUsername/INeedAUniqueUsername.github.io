LC-Sharp

Introduction

LC-Sharp is an advanced emulator for LC-3 assembly language. It provides all the functionality of the simpl emulator plus more. The program is written in C# using the Terminal.Gui library by Miguel de Icaza. LC-Sharp is developed by Alex Chen and Yourui Xue with supervision by Brian Linard.

Features

- Instruction view provides a scrollable list of all instructions with the Program Counter highlighted
- Register view provides a listing of all register values
- Label view provides a listing of all defined labels in the program
- Assembly field allows you to write code to the LC-3 on the fly.
- Running automatically stops upon encountering a NOP
- Internal state machine replicates that of the LC-3
- Assembler supports custom TRAP subroutines
