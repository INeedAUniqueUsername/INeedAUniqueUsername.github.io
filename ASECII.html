ASECII

Introduction

ASECII (ASCII Sprite Editing Console II) is a fully featured ASCII art painter inspired by Aseprite and REXPaint.

Features

- Brush (tile placement)
- Rectangle select
- Wand select (local / global)
- Freeform select (lasso / polygon)
- Move
- Color palette
- Layer management
- Save / load
